A widget to display Trunk.ly links

== Description ==

This plugin creates a widget that displays your latest Trunk.ly links

== Installation ==

1. Upload `trunkly` folder to your `/wp-content/plugins/` directory
1. Make sure the cache folder is set to 777
1. Activate the plugin through the 'Plugins' menu in WordPress
